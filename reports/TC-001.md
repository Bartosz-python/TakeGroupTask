*Summary: [UI][Functionality] "Clear" button lacks functionality in the sort feature in the Movies subtab

*Description: When navigating to the vod.film page and entering the Movies subtab, selecting the sorting option
and pressing a "Clear" button, the applied criteria are not resetted or removed.

This issue has been reproduced on Chrome and Firefox on Windows 11 (10.0.26100).

*Steps:
1. Launch the browser.
2. Navigate to the "https://vod.film/" URL.
3. Proceed to the movie subtab.
4. Open the "Sort" feature.
5. Choose any sorting criteria.
6. Press the "Clear" button.
7. Observe that the "Clear" button lacks functionality.

*Expected results: The "Clear" button should reset all sorting criteria to their default state, and the list should show unfiltered movies.

*Actual results: The "Clear" button has no effect. Applied sorting criteria remain active and movies continue to be filtered.

*Priority: High
